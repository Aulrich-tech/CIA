How to Use

Lessons 1–5 covers perimeter and application security: Firewalls, Network Access Control, Sandbox, WAF, and Secure Email Gateway.

Lessons 6–10 covers content filtering, wireless security, endpoint hardening, endpoint monitoring, and SOAR automation.

Lessons 11–15 covers SIEM, SD-WAN, Zero Trust Network Access, Cloud Security, and SASE.

SASE — Secure Access Service Edge. SSE + SD-WAN, SWG, FWaaS, CASB, ZTNA, centralized management.

Firewall Generations

1st Gen: Stateless — filters by IP, protocol, port

2nd Gen: Stateful — tracks 5-tuple & connection state

3rd Gen: Application layer filtering (HTTP, FTP, DNS)

NGFW: DPI + sandbox + AI + segmentation

Sandbox Generations

1st Gen: Standalone — poor integration & intel sharing

2nd Gen: Integrated — centralized threat correlation

3rd Gen: AI/ML-driven — proactive, MITRE ATT&CK, real-time

WAF Generations

1st Gen: Blocklists + signature-based detection

2nd Gen: Behavior baselines + heuristics + session monitoring

3rd Gen: Machine learning, DDoS defense, DLP, sandbox integration
